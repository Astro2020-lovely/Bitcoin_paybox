1. Download and extract files

Download plugin from your "Downloads" section in your CodeCanyon account. Then extract archive. Folders assets and classes are required to plugin work without problems.

2. How to use plugin

You can see example code in file bitcoin_example.php. 

Here is sample code:
<?php
require("config.php");
require("classes/block_io.php");
require("classes/gateway.php");

$box = new BtcGateway();
echo $box->create_payment_box("0.00012"); // where is 0.00012 type your requested Bitcoin amount.
?>

 

3. Payment verification

To verify client payment and process order you can check file bitcoin_ipn.php

If location or bitcoin_ipn.php file was renamed need to change also in assets/js/gateway.js file lines.

 

function btc_gateway_update_status(boxID) {
    var payment_div = $("#PaymentStatus_"+boxID);
    var data_url = "bitcoin_ipn.php"; // i mean here.
    $.ajax({
        type: "GET",
        url: data_url,
        dataType: "html",
        success: function (data) {
            payment_div.html(data);
        }
}
